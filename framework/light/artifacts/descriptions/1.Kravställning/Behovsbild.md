# Artefakt: Behovsbild

## Typ

Krav

## Beskrivning

Sammanhållen beskrivning av varför initiativet finns, vilka problem som ska lösas och vilket värde som ska uppnås.

Innehåller en strukturerad bild av:

- behov och problem
- målbild
- värde (på hög nivå)
- scope (ingår / ingår inte)
- stakeholders (på hög nivå)

Utgör grunden för att förstå VAD som ska lösas innan funktionalitet och prioritering definieras.
