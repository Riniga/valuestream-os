# Processer – De 5 Faserna i ValueStream OS

ValueStream OS organiserar arbetet i fem förbundna faser som upprepar sig i en cirkel.

## De 5 Processerna

1. **[Kravställning (WHAT)](./1.%20Kravställning.md)** – Vad behöver vi bygga?
2. **[Målarkitektur (HOW)](./2.%20Målarkitektur.md)** – Hur ska lösningen se ut?
3. **[Roadmap (WHEN)](./3.%20Roadmap.md)** – När ska det levereras?
4. **[Leverans (BUILD)](./4.%20Leverans.md)** – Hur bygger vi det?
5. **[Repeat (LEARN)](./5.%20Repeat.md)** – Vad lärde vi oss?
