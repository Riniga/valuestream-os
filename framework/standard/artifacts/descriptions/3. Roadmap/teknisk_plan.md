# Artefakt: Sekvensplan

## Typ
Plan

## Beskrivning
Beskriver tekniska beroenden och genomförandeordning för implementation av lösningen.

Artefakten används för att säkerställa att tekniska förutsättningar, integrationer och komponenter implementeras i rätt ordning och utan att blockera varandra.

Teknisk plan svarar på:
- vilka tekniska beroenden som finns mellan komponenter och funktioner
- i vilken ordning implementation bör ske
- vilka tekniska constraints som påverkar genomförandet
- vilka risker och blockerare som behöver hanteras

Artefakten är ett komplement till roadmap och används endast vid behov när teknisk komplexitet kräver fördjupad planering.
