# Artefakt: Epics & Capabilities

## Typ

Krav

## Beskrivning

Beskriver övergripande funktionalitet och förmågor som lösningen ska stödja.
Syftet är att strukturera och gruppera behov i större enheter som kan brytas ner till mer detaljerade krav och ligga till grund för planering och prioritering.
Artefakten fungerar som en brygga mellan övergripande behov och detaljerade user stories.
