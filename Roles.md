# Roller för att ta fram kravställning
| Roll | Ansvar |
|-----|------|
| Produktägare | Ansvarar för produktvision och prioritering av behov |
| Verksamhetsexperter | Bidrar med kunskap om processer och behov |
| Användarrepresentanter | Säkerställer användarperspektiv |
| Business Analyst | Strukturerar krav, analyser och dokumentation |
| Arkitekt | Säkerställer att krav är tekniskt genomförbara |
| UX / Service designer | Arbetar med användarresor och interaktionsflöden |
| Projektledare / Initiativledare | Koordinerar arbetet i fasen |
| Utvecklingsrepresentanter | Bidrar med teknisk förståelse och genomförbarhet |

## För att ta fram Målarkitektur
| Roll | Ansvar |
|-----|------|
| Lösningsarkitekt | Leder arkitekturarbetet och definierar målarkitekturen |
| Enterprise arkitekt | Säkerställer att lösningen följer organisationens arkitekturprinciper |
| Produktägare | Säkerställer att arkitekturen stödjer produktens behov |
| Business Analyst | Säkerställer koppling mellan krav och lösning |
| Teknisk lead | Bidrar med teknisk genomförbarhet |
| Säkerhetsarkitekt | Ansvarar för säkerhetsarkitektur |
| Dataarkitekt | Ansvarar för datamodell och informationsstruktur |
| DevOps | Säkerställer driftbarhet och teknisk plattform |
| UX | Säkerställer att arkitekturen stödjer användarflöden |

## Roller för att ta fram roadmap
| Roll | Ansvar |
|-----|------|
| Produktägare | Ansvarar för prioritering och värde |
| Verksamhetsrepresentanter | Säkerställer verksamhetsnytta |
| Lösningsarkitekt | Säkerställer teknisk genomförbarhet |
| Business Analyst | Strukturerar backlog och funktionalitet |
| Teknisk lead | Identifierar tekniska beroenden |
| Utvecklingsrepresentanter | Bidrar med estimat och teknisk förståelse |
| Projektledare | Koordinerar planering och roadmap |



# Roller för att leverera

| Roll | Ansvar |
|-----|------|
| Produktägare | Prioriterar backlog och säkerställer verksamhetsvärde |
| Scrum Master | Leder teamets arbetsprocess |
| Utvecklare | Implementerar funktionalitet |
| Testare | Säkerställer kvalitet genom test |
| DevOps | Ansvarar för deployment, drift och automatisering |
| UX | Säkerställer användbarhet |
| Lösningsarkitekt | Säkerställer att implementation följer målarkitektur |
| Business Analyst | Förtydligar krav och stödjer refinement |
| Verksamhetsrepresentanter | Deltar i demo och ger feedback |

# Roller för Repeat / Reflektion & Justering

| Roll | Ansvar |
|------|--------|
| Produktägare | Utvärderar uppnått värde, samlar feedback och skapar input till nästa kravställning |
| Verksamhetsrepresentanter | Ger verksamhetsfeedback på levererad funktionalitet och bekräftar värdeutfall |
| Scrum Master | Leder retrospektiv, analyserar teamets arbetssätt och driver förbättringar |
| Utvecklingsteam | Bidrar med tekniska lärdomar, identifierar hinder och förbättringsmöjligheter |
| Business Analyst | Strukturerar feedback till övergripande kravinsikter inför nästa cykel |
| Lösningsarkitekt | Identifierar arkitekturella konsekvenser och tekniska lärdomar som input till nästa målarkitektur |
| Teknisk Lead | Identifierar tekniska beroenden, risker och kvalitetsaspekter för nästa roadmap‑arbete |
| Projektledare | Säkerställer att insamlad input tas om hand, kommuniceras och används för att starta nästa cykel |
