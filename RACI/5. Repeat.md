# Roller nödvändiga för Repeat / Reflektion & Justering

```mermaid
flowchart LR

%% ===== SEKVENTIELL INGÅNG =====
A0["Start: Leverans avslutad"] --> A1["Utvärdera leveransresultat"]
A0 --> A2["Utvärdera process & arbetssätt"]

%% ===== PARALLELLA FÖRBEREDELSER INFÖR NÄSTA CYKEL =====
A1 --> B1["Samla input för uppdaterad kravställning"]
A2 --> B1
A1 --> B2["Identifiera arkitekturella lärdomar"]
A2 --> B2
A1 --> B3["Förbereda nästa roadmap‑iteration"]
A2 --> B3

%% ===== KONSOLIDERING OCH ÅTERSTART =====
B1 --> C1["Förbereda nästa cykel"]
B2 --> C1
B3 --> C1

C1 --> D0["Output: Input till 'Kravställning' (starta nästa cykel)"]

%% ===== STYLING =====
classDef seq fill:#1e88e5,stroke:#0d47a1,color:#fff;
classDef par fill:#43a047,stroke:#1b5e20,color:#fff;
classDef converge fill:#6a1b9a,stroke:#4a148c,color:#fff;

class A0,A1,A2 seq;
class B1,B2,B3 par;
class C1,D0 converge;
```
